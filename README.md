# Activities_RStudio

¡Bienvenido/a a mi repositorio de ejercicios y proyectos desarrollados en el entorno de RStudio!

Este espacio sirve como un portafolio digital y un registro de mi progreso en el aprendizaje y la aplicación práctica del lenguaje de programación R. Aquí se recopilan diversas actividades que cubren desde conceptos básicos hasta la implementación de análisis más complejos.

Propósito: Demostrar mis habilidades y comprensión de las funcionalidades de R y las herramientas de RStudio.

Contenido: Los directorios contienen archivos .R, .Rmd (R Markdown) u otros formatos relevantes, cada uno enfocado en una actividad o proyecto específico.
